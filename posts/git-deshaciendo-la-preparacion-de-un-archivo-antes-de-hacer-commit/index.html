Git - Deshaciendo la preparación de un archivo antes de hacer commit

Manuel Villaverde

2015-04-23 22:53

Código fuente

Cuando por error añadimos un archivo al área de preparación para hacer un comiit podemos quitarlo con reset HEAD archivo

Pongo un ejemplo

$ git add readme.txt

$ git status

# On branch master

# Changes not staged for commit:

# (use "git reset HEAD <file>..." to unstage)

#

#    modified:   readme.txt

#
